# Wedding Management
>It's a mini project for database management system(DBMS) using MySql , basically for wedding management

### Technology used :
         
- phpmyadmin used for mysql database

- php used for backend

- html5 | css | javascript | bootstrap4 for frontend
          
- Xampp server

### Guidlines :

step 1:- Install Xampp 

step 2:- Open Xampp folder and  paste the whole project inside htdocs folder

step 3:- Start Xampp server and start apache and mysql

step 4:- Open phpmyadmin using this link http://localhost/phpmyadmin/ and create a database with name-'wedding'. Name should be same otherwise it shows error

step 5:- Inside the project folder there's a file named wedding.sql, import it in phpmyadmin inside the wedding database

step 6:- Check your project http://localhost/wedding/
